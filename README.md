# AI MCQ Generator

This project is an AI-powered application that generates multiple-choice questions from study material.

## Features
- Upload text or study material
- AI analyzes the content
- Generates MCQs automatically
- Simple UI built with Streamlit

## Technologies Used
- Python
- Streamlit
- LLM API integration

## Note
To run this project, you need to provide your own OpenAI API key.

Set it as an environment variable:

OPENAI_API_KEY=your_api_key_here
